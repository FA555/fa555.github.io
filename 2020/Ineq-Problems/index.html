设 $a, b, c > 0, \ a^2 b^3 c = 9$。求 $2a + b + 3c$ 的最小值。

答案 $6$。

解：$2a + b + 3c = a + a + \frac b3 + \frac b3 + \frac b3 + 3c \ge 6 \sqrt[6]{ \frac{a^2 b^3 c}9} = 6$，当且仅当 $a = \frac b3 = 3c$ 时取等号。故 $2a + b + 3c$ 的最小值为 $6$，当 $a = 1, b = 3, c = \frac 13$ 时取到。

设 $a, b, c > 0$。求证：$abc \ge (b + c - a)(a + c - b)(a + b - c)$。

这题可以直接平方差讨论，头铁可以用均值不等式。

证明：$a = \frac{a + b - c}2 + \frac{a + c - b}2 \ge \sqrt{(a + b - c)(a - b + c)}$。$b, c$ 同理。三个式子相乘即得原不等式。$\square$

设 $a, b, c > 0, \frac a{1 + a} + \frac b{1 + b} + \frac c{1 + c} = 1$。求证：$\frac 1{a^2} + \frac 1{b^2} + \frac 1{c^2} \ge 12$。

证明：令 $x = \frac 1a, y = \frac 1b, z = \frac 1c$，则 $\frac 1{1 + x} + \frac 1{1 + y} + \frac 1{1 + z} = 1$，原命题即证 $x^2 + y^2 + z^2 \ge 12$。

  • 构造 $\frac 1{1 + x} + \frac{1 + x}9 \ge \frac 23$，当且仅当 $x = 2$ 时取等号。$y$ 和 $z$ 同理。三式相加得 $x + y + z \ge 6$。$(x + y + z)^2 \ge 36$，又有 $x^2 + y^2 \ge 2xy$，代入得 $3(x^2 + y^2 + z^2) \ge 36$。$\square$
  • 柯西不等式。

设 $a, b, c > 0, a + b + c = 1$，求 $\sqrt{15a + 1} + \sqrt{15b + 1} + \sqrt{15c + 1}$ 的最大值。

答案 $3 \sqrt 6$。

解：$\sqrt{15a + 1} + \sqrt{15b + 1} + \sqrt{15c + 1} \le \sqrt 3 \times \sqrt{15(a + b + c) + 3} = 3 \sqrt 6$，当且仅当 $a = b = c = \frac 13$ 时取等号。

设 $a, b, c > 0, a + b + c = 1$，求 $\sqrt{a + b} + \sqrt{b + c} + \sqrt{a + c}$ 的最大值。

答案 $\sqrt 6$。

解：$\sqrt{a + b} + \sqrt{b + c} + \sqrt{a + c} \le \sqrt 3 \times \sqrt{2(a + b + c)} = \sqrt 6$，当且仅当 $a = b = c = \frac 13$ 时取等号。

设 $a, b, c > 0, a + b + c = 1$，求 $a^2 + b^2 + c^2$ 的最小值。

答案 $\frac 13$。

解：$a^2 + b^2 + c^2 \ge \frac{(a + b + c)^2}3 = \frac 13$，当且仅当 $a = b = c = \frac 13$ 时取等号。

设 $a, b, c > 0, a + b + c = 1$，求 $\frac 1a + \frac 1b + \frac 1c$ 的最小值。

答案 $9$。

解：$\frac 1a + \frac 1b + \frac 1c \ge \frac{3^2}{a + b + c} = 9$，当且仅当 $a = b = c = \frac 13$ 时取等号。

设 $a, b, c > 0, a + b + c = 1$，求 $\frac 1a + \frac 4b + \frac 9c$ 的最小值。

答案 $36$。

解：$\frac 1a + \frac 4b + \frac 9c \ge \frac{(1 + 2 + 3)^2}{a + b + c} = 36$，当且仅当 $\frac 1a = \frac 2b = \frac 3c$，即 $a = \frac 16, b = \frac 13, c = \frac 12$ 时取等号。

设 $a, b, c, d, e > 0, a + b + c + d + e = 8, a^2 + b^2 + c^2 + d^2 + e^2 = 16$，求 $e$ 的最大值。

答案 $\frac{16}5$。

解：$a^2 + b^2 + c^2 + d^2 \ge \frac{(a + b + c + d)^2}4$，即 $16 - e^2 \ge \frac{(8 - e)^2}4$。解得 $0 \le e \le \frac{16}5$。

三边长为 $a, b, c$ 的 $\triangle ABC$ 内有一点 $P$，$P$ 到 $a, b, c$ 三边的距离分别为 $x, y, z$，求 $x^2 + y^2 + z^2$ 的最小值。

答案 $\frac{(a + b + c)(b + c - a)(a + c - b)(a + b - c)}{4(a^2 + b^2 + c^2)}$。

解：设三角形面积为 $S$，显然可以用 $a, b, c$ 表示。有 $S = \frac 12(ax + by + cz)$。$(x^2 + y^2 + z^2)(a^2 + b^2 + c^2) \ge (ax + by + cz)^2 = 4S^2$，所以 $x^2 + y^2 + z^2 \ge \frac{4S^2}{a^2 + b^2 + c^2}$，当且仅当 $\frac xa = \frac by = \frac cz$ 时取等号。

设 $n \in \N, n \ge 2$。求证：$\frac 47 < \frac 1{n + 1} + \frac 1{n + 2} + \cdots + \frac 1{2n} < \frac{ \sqrt 2}2$。

证明：

  • 对于左半部分，$\mathrm{R.H.S.} \ge \frac{n^2}{n + 1 + n + 2 + \cdots + 2n} = \frac{2n}{3n + 1} \ge \frac 47$。又因为 $\frac 1{n + 1} = \frac 1{n + 2} = \cdots + \frac 1{2n}$ 不可能成立，所以等号不可能取到。

  • 对于右半部分，不等式两边平方得 $\left( \frac 1{n + 1} + \frac 1{n + 2} + \cdots + \frac 1{2n} \right)^2 < \frac 12$。

    $\square$
