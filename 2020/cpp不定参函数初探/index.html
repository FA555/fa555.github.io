va_list 系 (C)

示例

优点

C 语言只能用这个，就没啥别的优点了（

这个在 C 库的应用比较广泛，所以应该熟练掌握如何调用（大雾

缺点

继承了 C 的各种缺点。面向内存管理，导致类型转换非常麻烦，容易出错和越界。

必须传一个参数能够推断出参数类型和个数，非常麻烦。比如 printf 就要传一个 const char*，用格式符推断类型和个数。

格式十分严格，比如 va_end 不加就会引发内存问题。
