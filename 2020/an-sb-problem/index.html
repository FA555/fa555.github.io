一道 sb 题
2020-03-15
字数总计: 711|阅读时长: 3 分钟

problem

Alice 和 Bob 在一组排列成长条型的 $n$ 个格子上玩游戏，两人交替进行操作。

每回合中，一个玩家选择两个连续的空白格子，然后把它们涂黑。

最先不能进行操作的玩家输。Alice 首先进行操作。

  • 如果 $n = 1$，没有合法操作，Alice 直接输掉。

  • 如果 $n = 2$，只有一种合法操作，Alice 操作后 Bob 输。

  • 如果 $n = 3$，Alice 有两种合法操作，但不论哪种操作过后 Bob 都会输。

  • 如果 $n = 4$，Alice 有三种合法操作，他可以涂黑最中间的两个格子使 Bob 无法操作。

  • 如果 $n = 5$，如图，Alice 有四种合法操作（红色）；但无论他怎么操作，Bob（蓝色）都会赢。

因此，对于 $1 \le n \le 5$ 的情况，有 $3$ 种 $n$ 的取值使得 Alice 必胜。

类似地，对于 $1 \le n \le 50$ 的情况，有 $40$ 种 $n$ 的取值使得 Alice 必胜。

显然 Bob 非常想让 Alice 赢，但他不想放水。

假设 Bob 和 Alice 都绝对聪明，永远会采取最优策略。

Bob 想知道：对于 $1 \le n \le 1000000$ 的情况，有多少种 $n$ 的取值使得 Alice 必胜？

solution

不用动脑子的打表做法。

博弈论问题，直接考虑要求出每个 $n$ 的 $\mathrm{sg}$ 函数。

枚举所有可能的操作（将原游戏分为大小为 $j$ 和 $i - j - 2$ 的两个游戏），容易得到这样一个式子：

由 $n \in \{0, 1\}$ 时先手必败，得到 $\mathrm{sg}(0) = \mathrm{sg}(1) = 0$。
