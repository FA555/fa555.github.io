一道 sb 题
2020-03-15
字数总计: 713|阅读时长: 3 分钟

problem

小葛和小巩在一组排列成长条型的 $n$ 个格子上玩 游 戏，两人交替进行操作。

每回合中，一个玩家选择两个连续的空白格子，然后把它们涂黑。

最先不能进行操作的玩家输。小葛首先进行操作。

  • 如果 $n = 1$，没有合法操作，小葛直接输掉。

  • 如果 $n = 2$，只有一种合法操作，小葛操作后小巩输。

  • 如果 $n = 3$，小葛有两种合法操作，但不论哪种操作过后小巩都会输。

  • 如果 $n = 4$，小葛有三种合法操作，他可以涂黑最中间的两个格子使小巩无法操作。

  • 如果 $n = 5$，如图，小葛有四种合法操作（红色）；但无论他怎么操作，小巩（蓝色）都会赢。

因此，对于 $1 \le n \le 5$ 的情况，有 $3$ 种 $n$ 的取值使得小葛必胜。

类似地，对于 $1 \le n \le 50$ 的情况，有 $40$ 种 $n$ 的取值使得小葛必胜。

由于小葛是直男，他非常想赢过小巩。

他想知道：对于 $1 \le n \le 1000000$ 的情况，有多少种 $n$ 的取值使得他必胜？

solution

原题链接 Project Euler Problem 306 - Paper-strip Game

我连数据范围都没改

不用动脑子的打表做法。

博弈论问题，直接考虑要求出每个 $n$ 的 $\mathrm{sg}$ 函数。

枚举所有可能的操作（将原游戏分为大小为 $j$ 和 $i - j - 2$ 的两个游戏），容易得到这样一个式子：

由 $n \in \{0, 1\}$ 时先手必败，得到 $\mathrm{sg}(0) = \mathrm{sg}(1) = 0$。

打表可得 $1 \le n \le 204$ 时的 $\mathrm{sg}$ 函数值（每行 $34$ 个）：

容易发现除前两行特殊（共 $13$ 个 $0$）以外，以后的每一行都有恰好 $5$ 个 $0$。
