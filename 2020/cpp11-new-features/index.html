前言

C++11 是 C++ 编程语言的第二个主要版本，是自 C++98 以来的最重要的更新。

相比于 C++03，C++11 标准包含核心语言的新机能，而且扩展 C++ 标准程序库，现已被国内大部分算法竞赛所采用。

本文将介绍 C++11 中部分新的语法和标准库在算法竞赛中的实际应用。

空指针

C++ 并不像 C 一样允许将 void* 隐式地转换为其他类型的指针，而是建议 NULL 应当定义为 0，因此 foo(NULL) 将会调用 foo(int)。

标准库

散列容器

C++11 提供了四种新的散列容器，分别是 std::unordered_set, std::unordered_multiset, std::unordered_map, std::unordered_multimap，他们都满足容器、具分配器容器、无序关联容器的要求。

他们均支持插入、擦除、查找元素。类似其他散列容器，他们的平均复杂度均为常数，最坏复杂度均与容器大小成线性。

正则表达式

C++11 添加了新的程序库用于处理正则表达式，包含于 头文件。

std::array 类

std::array 是封装固定大小数组的容器。

此容器是一个聚合类型，其语义等同于保有一个 C 风格数组 T[N] 作为其唯一非静态数据成员的结构体。不同于 C 风格数组，它不会自动退化成 T* 。

算法库函数

std::all_of, std::any_of, std::none_of

检查一元谓词 p 是否对范围 [first, last) 中所有元素返回 true。

检查一元谓词 p 是否对范围 [first, last) 中至少一个元素返回 true。

检查一元谓词 p 是否不对范围 [first, last) 中任何元素返回 true。

std::minmax_element

寻找范围 [first, last) 中最小和最大的元素。用 operator< 比较元素。

返回 std::pair，其中前一个元素是指向最小值的迭代器，后一个元素是指向最大值的迭代器。

std::iota

以始于 value 并重复地求值 ++value 的顺序递增值填充范围 [first, last)。
