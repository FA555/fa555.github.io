这里是暑假化学一轮笔记。

由于我化学非常菜，我会写所有我觉得有用的东西。

可以明显感觉到我不像喜欢数学那样喜欢化学。学数学是乐趣，学化学是任务。

[TOC]

化学计量

  • $n = \frac N{N_A} = \frac mM = \frac V{V_\textrm m} = c V_\textrm{aq}$
  • $pV = nTR$
    • $T, p$ 相同，则 $MV, \frac Vn, \frac mM, \frac M \rho$ 相同，进而得到 $\frac m \rho$ 相同。
    • $T, V$ 相同，则 $Mp, \frac M \rho$ 相同。

化学物质及其变化

物质的分类

  • 导电的条件：带电微粒；自由移动的环境。导电能力与带电微粒的浓度有关。

分散系与胶体

  • 分散系由分散剂和分散质构成。分散质粒子直径小于 $1 \ \pu{nm}$ 的为溶液；$1 \ \pu{nm} \sim 100 \ \pu{nm}$ 之间的为胶体，$100 \ \pu{nm}$ 以上的为浊液。
  • 胶体的物理性质：丁达尔效应；电泳现象；介稳性；聚沉现象；胶体粒子不能透过半透膜，但能透过滤纸。
    • 区分胶体和其他分散系的最简便方法是利用丁达尔效应。
    • 胶体不带电，某些胶体微粒吸附带电微粒而带电。胶体电泳现象的原因是胶体粒子带有相同的电荷。淀粉胶体粒子不带电，所以没有电泳现象。
    • 胶体微粒一般是粒子、分子或难溶物的聚集体。因此，$\ce{Fe(OH)3}$ 胶体中 $\ce{Fe(OH)3}$ 胶体粒子的数目远远小于原 $\ce{FeCl3}$ 溶液中 $\ce{Fe^3+}$ 的数目。
    • 向 $\ce{Fe(OH)3}$ 胶体中加入强酸，先出现沉淀，后沉淀溶解。出现沉淀是因为向胶体溶液中加入电解质发生聚沉；沉淀溶解是因为发生了中和反应。
  • 常见胶体
    • 气溶胶：烟、云、雾。
    • 液溶胶：$\ce{Fe(OH)3}$ 胶体、$\ce{Al(OH)3}$ 胶体、硅酸胶体、淀粉胶体、蛋白质胶体、豆浆、墨水、涂料、肥皂水。
    • 固溶胶：烟水晶、有色玻璃。

离子共存

  • 离子不能大量共存的几种类型
    • 发生反应生成沉淀：$\ce{H+}$ 与 $\ce{SiO3^2-}$；$\ce{OH-}$ 与 $\ce{Mg^2+, Al^3+, Cu^2+, Fe^2+, Fe^3+}$ 等；$\ce{Ag+}$ 与 $\ce{Cl-, Br-, I-}$ 等；$\ce{SO4^2-, CO3^2-}$ 与 $\ce{Ba^2+, Ca^2+, Ag+}$ 等。
    • 发生反应生成弱电解质：$\ce{H+}$ 与 $\ce{OH-, CH3COO-, F-, ClO-}$ 等。
    • 发生反应生成气体：$\ce{H+}$ 与 $\ce{CO3^2-, HCO3-, SO3^2-}$ 等；加热条件下 $\ce{OH-}$ 与 $\ce{NH4+}$ 等。
    • 发生氧化还原反应：酸性条件下 $\ce{MnO4-, NO3-}$ 与 $\ce{Fe^2+, I-}$ 等。
    • 发生相互促进的水解反应：$\ce{Al^3+, Fe^3+}$ 与 $\ce{AlO2-, HCO3-}$ 等。
    • 发生络合反应：$\ce{Fe^3+}$ 与 $\ce{SCN-}$。
  • 检验 $\ce{SO4^2-}$：向待测液中加足量盐酸酸化，取清液滴加 $\ce{BaCl2}$ 溶液，产生白色沉淀，则原溶液中含有 $\ce{SO4^2-}$。
  • 检验 $\ce{CO3^2-}$：向待测液中加 $\ce{BaCl2}$（或 $\ce{CaCl2}$）溶液，产生白色沉淀；再加入盐酸，产生无色无味气体；将所得气体通入澄清石灰水，石灰水变浑浊，则待测液中含有 $\ce{CO3^2-}$。
  • 检验 $\ce{Fe^2+}$
    • 若溶液中只含 $\ce{Fe^2+}$，用 $\ce{KSCN}$ 溶液和氯水检验。
    • 若溶液中含有 $\ce{Fe^2+, Fe^3+}$，用 $\ce{KMnO4}$ 溶液检验。
    • 若溶液中含有 $\ce{}$$\ce{Fe^2+, Fe^3+, Cl-}$，加入 $\ce{K3[Fe(CN)6]}$ 溶液，生成蓝色沉淀说明原溶液中含 $\ce{Fe^2+}$。

氧化还原反应

  • $\ce{K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au}$

    单质的还原性逐渐减弱，对应阳离子的氧化性逐渐增强。

  • $\ce{F Cl Br I S}$

    单质的氧化性逐渐减弱，对应阴离子的还原性逐渐增强。

  • 氧化性：$\ce{MnO4-} > \ce{Cl2} > \ce{Br2} > \ce{Fe^3+} > \ce{I2}$

    还原性：$\ce{S^2-} > \ce{SO3^2-} > \ce{I-} > \ce{Fe^2+} > \ce{Br-} > \ce{Cl-}$

金属及其化合物

钠的主要性质及其应用、碱金属元素

钠

  • 物理性质

    银白色，有金属光泽。熔点低于 $100 \pu{^\circ C}$。质地软。$\rho_\ce{H2O} > \rho_\ce{Na} > \rho_{ \mbox{煤油}}$。

  • 化学性质

  • 制取

    $\ce{2 NaCl \text{（熔融）} \xlongequal{ \mbox{通电}} 2Na + Cl2 ^}$

  • 保存

    密封保存在石蜡油或煤油中。不能保存在 $\ce{CCl4}$ 中，因为 $\ce{CCl4}$ 的密度比钠大。

焰色反应

属于元素的物理性质。

$\ce{Na}$ 焰色为黄色。$\ce{K}$ 焰色为紫色，且观察时要透过蓝色钴玻璃。

碱金属

除 $\ce{Cs}$ 略带金色光泽外，都是银白色有金属光泽的固体，密度较小，硬度较小，熔、沸点较低，导电性、导热性较强。

从 $\ce{Li}$ 到 $\ce{Cs}$ 密度增大，熔、沸点降低，金属性增强。有一个特例是 $\ce{Na}$ 的密度大于 $\ce{K}$。

都能与 $\ce{O2}$ 发生反应。$\ce{4Li + O2 \xlongequal{ \mbox{点燃}} 2Li2O}$，而 $\ce{K, Rb, Cs}$ 与 $\ce{O2}$ 在点燃时反应生成比过氧化物结构更复杂的物质。都与水反应生成碱和 $\ce{H2}$，从 $\ce{Li}$ 到 $\ce{Cs}$ 反应越来越剧烈。

一般都保存在煤油中。$\ce{Li}$ 的密度小于煤油，所以保存在石蜡中。

都与 $\ce{H2}$ 反应生成强还原剂、离子化合物 $\ce{RH}$，其中 $\ce{R}$ 是碱金属，$\ce{H}$ 显 $-1$ 价。

钠的氧化物

$\ce{Na2O}$ $\ce{Na2O2}$
颜色、状态 白色固体 淡黄色固体
类别 碱性氧化物 过氧化物
热稳定性 $\ce{2Na2O + O2 \xlongequal{ \triangle} 2Na2O2}$ 稳定
与水反应 $\ce{Na2O + H2O \xlongequal{} 2NaOH}$ $\ce{2Na2O2 + 2H2O \xlongequal{} 4NaOH + O2 ^}$
与 $\ce{CO2}$ 反应 $\ce{Na2O + CO2 \xlongequal{} Na2CO3}$ $\ce{2Na2O2 + 2CO2 \xlongequal{} 2Na2CO3 + O2}$
与盐酸反应 $\ce{Na2O + 2HCl \xlongequal{} 2NaCl + H2O}$ $\ce{2Na2O + 4HCl \xlongequal{} 4NaCl + 2H2O + O2 ^}$
用途 制取 $\ce{Na2O2, NaOH}$ 强氧化剂、漂白剂、供养剂、消毒剂
  • $\ce{Na2O2}$ 强氧化性的表现
    • $\ce{Na2O2 + SO2 \xlongequal{} Na2SO4}$
    • $\ce{4Na2O2 + 4FeCl2 + 6H2O \xlongequal{} 4Fe(OH)3 v + O2 ^ + 8NaCl}$
    • $\ce{Na2O2 + H2S \xlongequal{} S v + 2NaOH}$
    • 使酚酞溶液先变红后褪色
    • 使品红溶液褪色
  • $\ce{Na2O2}$ 遇 $\ce{KMnO4}$ 等强氧化剂显还原性，氧化产物 $\ce{O2}$；遇 $\ce{CO2, H2O, H+}$ 发生自身氧化还原反应。

碳酸钠和碳酸氢钠

$\ce{Na2CO3}$ $\ce{NaHCO3}$
俗名 纯碱、苏打 小苏打
颜色、状态 白色晶体 细小白色晶体
热稳定性 稳定 $\ce{2NaHCO3 \xlongequal{ \triangle} Na2CO3 + CO2 ^ + H2O}$
与 $\ce{H+}$ 反应 $\ce{Na2CO3 + 2H+ \xlongequal{} 2Na+ + CO2 ^ + H2O}$ $\ce{NaHCO3 + H+ \xlongequal{} Na+ + CO2 ^ + H2O}$
与 $\ce{Ca(OH)2}$ 反应 $\ce{Ca^2+ + CO3^2- \xlongequal{} CaCO3 v}$ $\ce{Ca^2+ + OH- + HCO3- \xlongequal{} CaCO3 v + H2O}$（碱过量）
$\ce{Ca^2+ + 2OH- + 2HCO3- \xlongequal{} CO3^2- + H2O + CaCO3 v}$（碱不足）
与 $\ce{Al2(SO4)3}$ 反应 $\ce{2Al^3+ + 3CO3^2- + 3H2O \xlongequal{} 2Al(OH)3 v + 3CO2 ^}$ $\ce{Al^3+ + 3HCO3- \xlongequal{} Al(OH)3 v + 3CO2 ^}$
用途 制玻璃、造纸、制皂、洗涤 发酵、医药、灭火器

  • $\ce{NaHCO3}$ 在水中的溶解度小于 $\ce{Na2CO3}$。因此，将 $\ce{CO2}$ 通入饱和 $\ce{Na2CO3}$ 溶液中，有沉淀析出。

  • | 混合物（杂质） | 除杂方法 |
    | —————————————————- | ————————————- |
    | $\ce{Na2CO3}$ 固体（$\ce{NaHCO3}$） | 加热 |
    | $\ce{Na2CO3}$ 溶液（$\ce{NaHCO3}$） | 加入适量 $\ce{NaOH}$ 溶液 |
    | $\ce{NaHCO3}$ 溶液（$\ce{Na2CO3}$） | 通入足量 $\ce{CO2}$ 气体 |

  • $\ce{Na2CO3}$ 的制备：侯氏制碱法

    • 原理：向饱和 $\ce{NaCl}$ 溶液中通入 $\ce{CO2}$ 和 $\ce{NH3}$ 气体，使 $\ce{NaHCO3}$ 析出；过滤得到 $\ce{NaHCO3}$ 晶体，受热分解生成 $\ce{Na2CO3}$。
    • $\ce{NaCl + NH3 + CO2 + H2O \xlongequal{} NaHCO3 v + NH4Cl}$

  • $\ce{NaOH}$ 溶液与 $\ce{CO2}$ 的反应

    原理：$\ce{NaOH ->[CO2] Na2CO3 ->[CO2] NaHCO3}$

    | $x = n( \ce{CO2}) : n( \ce{NaOH})$ | 反应后溶质成分 |
    | ————————————————— | ——————————- |
    | $x < 1 / 2$ | $\ce{NaOH, Na2CO3}$ |
    | $x = 1 / 2$ | $\ce{Na2CO3}$ |
    | $1 / 2 < x < 1$ | $\ce{Na2CO3, NaHCO3}$ |
    | $x \ge 1$ | $\ce{NaHCO3}$ |

镁及其重要化合物的性质

镁

  • 物理性质

    具有银白色金属光泽的固体，密度、硬度均较小，熔点较低，有良好的导电性、导热性和延展性。

  • 化学性质

镁的重要化合物

  • $\ce{MgO}$
    • 碱性氧化物
  • $\ce{Mg(OH)2}$
    • 中强碱
    • 难溶于水
    • 溶解度小于 $\ce{MgCO3}$：$\ce{MgCO3 + H2O \xlongequal{ \triangle} Mg(OH)2 + CO2 ^}$

杂项

  • 易错的标准状况下非气态的物质：$\ce{SO3, HF}$（沸点约 $20 \pu{ ^\circ C}$）、$\ce{NO2}$（沸点约 $21 \pu{^\circ C}$）、$\ce{CH2Cl2, CHCl3, CCl4, C2H5OH}$、含大于等于 $5$ 个 $\ce{C}$ 原子的烃类。

  • 一些试剂的保存：$\ce{NaOH}$ 溶液用橡胶塞；氯水和浓硝酸用棕色试剂瓶和玻璃塞；氢氟酸用塑料瓶。
