帮岳立冬小鸽鸽搞的二分题目
2019-12-02|杂项
字数总计: 233|阅读时长: 1 分钟

题目背景

这是一道信息学的题目。

题目描述

有形如： $f(x) = ax + b$ 的一个一次函数，已知该函数各项的系数 $a, b$。现给定区间 $[l, r]$，保证 $\exists x \in [l, r]$ 使得 $f(x) = 0$。请你求出这个 $x$ 的值。

保证 $a, b \in \mathbb{R}$ 且 $-10^6 \le a, b \le 10^6$。

分析

有一个显然的事实：若 $\exists x_1, x_2$ 使得 $f(x_1) \times f(x_2) < 0$，则必定 $\exists x \in (x1, x2)$ 使得 $f(x) = 0$。
